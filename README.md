## Description

## Installation

```bash
$ npm install
```
## Running the app

```bash
# swagger url
http://localhost:3000/

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
